and d is the distance between (x1, y1) and (x2, y2).

The mode in mathematics/statics refers to most frequently occurring
element in a collection of elements
eg: in a collection of numbers [5, 8, 4, 2, 3, 3, 8, 3, 1, 3],
the mode is 3
, as the number 3 appeared most number of times(4 times) in the collection.

The goal is to first calculate the Euclidean distances between two given points,
provided as a list of X-Y coordinates and finally find the mode of this distance list
and return the value

Assume a list of point coordinates represented as list of list
((-11, 3), (8, 5), (-3, 2), (9,17)) The Euclidean distance between
two  points in the array represented by two values
(x0,y0) and (x1,y1) at the index location 0 and 1 respectively is
d=root((8 - -11)^2 +(5-3)^2 ) which results to 19 (taking the floor value).
similarly considering the Eucledean distance of other
pairs we get the array of distances as [19, 8, 24, 11 ,12 ,19].
Here 19 appears most hence the mode is [19].

Note:In case the answer in multiple numbers the order should be Ascending order.
eg: if the distance array is [19,11,11,19],
the mod should be [11,19].
Also if the input does not satisfy the constraints the function should return [-1]
Eg:[(-11,8) , (3,5,2)] //failed criteria since input contains multiple dimension.

The function "findModeArrray" accepts a list of 2D coordinates and returns a list of mode.
